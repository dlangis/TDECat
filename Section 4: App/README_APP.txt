In order to use the app, you need to download all the files from this section in the same folder and unzip the .zip files.
The following libraries are necessary in order to run the script : dash, dash-bootstrap-components, pandas, plotly, and numpy. 
After completed the aforementioned steps, you can run the .py file. The url for the app is: http://127.0.0.1:8000/ (you can Shift+click in the terminal).
In the app you can find all the information for the sources, along with a search bar in order to simultaneously plot the multi-wavelength photometry and the spectra.
